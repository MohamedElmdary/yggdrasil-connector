# Yggdrasil Connector

A desktop app for linux using fyne allows to easily work with yggdrasil.

## Download

```
$ wget https://github.com/MohamedElmdary/yggdrasil-connector/blob/master/yggdrasil-connector.tar.gz
```

## Run

1. Extract file
2. Install

```
$ sudo make install
```

## Notes

This app requires `root` user.
